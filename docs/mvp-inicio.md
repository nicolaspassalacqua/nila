# NILA MVP 1.0 (Inicio)

Objetivo tecnico inicial:
- Backend Django + DRF modular multi-tenant por `tenant_id`.
- Frontend Next.js (web responsive) con vistas cliente/profesional.
- Docker Compose para desarrollo local.

Alcance inicial implementado:
- Auth basica (registro/login JWT).
- Tenants y membresias por rol.
- Servicios (marketplace) por tenant.
- Clientes (CRM basico) por tenant.
- Turnos (booking) por tenant.

Siguiente iteracion:
- Waitlist automatica por cancelacion.
- WhatsApp templates y cola de mensajes.
- POS y cash flow basico.
