# Manual De Usuario

Estado: `MVP 1.1-alpha`

## 1. Perfiles
- Cliente/Paciente.
- Profesional/Empresa.

## 2. Flujo Cliente (MVP)
1. Registrarse e iniciar sesion.
2. Ver profesionales/servicios disponibles en su tenant.
3. Reservar turno.
4. Quedar en lista de espera para una franja/servicio.
5. Recibir oferta si se libera un turno.

## 3. Flujo Profesional (MVP)
1. Crear tenant (empresa/perfil).
2. Asignar memberships.
3. Crear servicios.
4. Crear clientes.
5. Crear/cancelar turnos.
6. Gestionar waitlist y ofertas.
7. Gestionar productos, ordenes, pagos y caja (cash movements).
8. Configurar plantillas de mensajes y cola de notificaciones.

## 4. Modulos activos
- Marketplace basico.
- CRM de clientes.
- Agenda/turnos.
- Waitlist por cancelacion.
- Notificaciones base.
- POS basico + caja.

## 5. Accesibilidad
- Interfaz limpia y legible.
- Botones visibles en acciones principales.
- Estructura orientada a baja friccion para usuario mayor.

## 6. Proxima fase funcional
- Confirmacion automatica completa de waitlist.
- Envio WhatsApp real.
- Facturacion avanzada.
